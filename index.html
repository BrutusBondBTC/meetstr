About

Riverside Bitcoin is a group focused on promoting the use of bitcoin and freedom tech.

The idea is for Bitcoiners to get to know each other in Cyberspace. The pleb who created this website is very bullish on nostr and learning to create a nostrchat-telegram bridge, but it's very early so if you have any questions, please see the Telegram channel. The meetups will be posted on The Telegram Group and The Orange Pill App. Nostr uses cryptography, but is not an investment. There is no token. There is no blockchain--Only private/public keypairs and relays. I have also decided to use the Meetup.com because most of the world has not yet used public/private key cryptography.

The mission of this group is to create a local online beacon like Gigi describes on the website Twenty One World in Riverside, California. Then, we can meet in meat space.
